# Steve Ano - MCA Busali Ward 2027 Campaign Website

## Overview

Professional campaign website for Steve Ano's MCA (Member of County Assembly) candidacy in Busali Ward under the UDA party.

**Live Site:** https://steveano2027.github.io

## Features

✅ **Professional Hero Section** with countdown timer & call-to-action buttons
✅ **Sticky Navigation Menu** with mobile responsiveness
✅ **About Steve** section with background & vision
✅ **6-Point Manifesto** with interactive modal details
✅ **Campaign Progress Tracker** showing priority completion rates
✅ **News & Updates Section** with latest campaign news
✅ **Photo Gallery** placeholder for community engagement photos
✅ **Testimonials** from real Busali residents
✅ **Volunteer Registration Form** with village/occupation tracking
✅ **Issue Reporting System** for community problem reporting
✅ **Newsletter Subscription** to stay connected
✅ **Contact Information** with multiple channels
✅ **Social Media Integration** (Facebook, WhatsApp, TikTok, YouTube)
✅ **Mobile Responsive Design** - works on all devices
✅ **Language Selector** (English/Kiswahili ready)
✅ **UDA Branding** - Official party colors and styling

   - Replace with: Steve's actual WhatsApp number

2. **Phone Number**:

   - Replace with: Steve's actual phone number

3. **Email Address**:
   - Find: `steveano.busali@gmail.com`
   - Update if needed

### Optional Enhancements

1. **Add Steve's Photo**:
   - Upload the photo to `assets/images/steve-hero.jpg` (recommended) or use an external image URL.
   - Replace the hero emoji `👨‍💼` in your `index.html` hero section with a Markdown or HTML image reference. Example Markdown to place in README or other docs:

     ![Person speaking into a microphone at an outdoor event](/assets/images/steve-hero.jpg)

   - For better performance and responsiveness, prefer a `<picture>` element with `srcset` and WebP fallbacks in your `index.html` (see site code suggestions in the project notes).

2. **Add Real Gallery Photos**:
   - Replace placeholder emojis in the gallery section with actual images from Steve's community work

3. **Update Social Media Links**:
   - Replace Facebook, TikTok, YouTube URLs with actual Steve Ano accounts

4. **Connect Forms to Database**:
   - Integrate volunteer form with Google Sheets (via Google Apps Script)
   - Setup newsletter system (Mailchimp, ConvertKit, etc.)
   - Connect issue reporting to admin dashboard

## Deployment

This site is automatically deployed on GitHub Pages:

**URL:** `https://steveano2027.github.io`

### How It Works

1. This repository is set to deploy from the `main` branch
2. Any changes pushed to `main` are automatically published
3. Updates go live in 1-3 minutes

### To Update the Site

1. Edit `index.html` with your changes
2. Commit and push to GitHub
3. Wait 1-3 minutes for the site to refresh
